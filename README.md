# CodeLab Revival


Live Platform: [CodeLab Revival Platform](https://codelabrevival.vercel.app/?utm_source=chatgpt.com)

**“Rebuild. Recode. Revive.”**

CodeLab Revival is a modern, full-stack coding education platform designed to improve the programming learning experience for both students and instructors. Developed as part of Brooklyn College’s CISC 4900 capstone course, the project reimagines the traditional CodeLab system with a focus on usability, intelligent feedback, accessibility, and scalability.

The platform features a redesigned responsive interface, an integrated browser-based coding environment, real-time error feedback, automated grading, progress tracking, and instructor analytics dashboards. Students receive contextual debugging hints for syntax, logic, and runtime errors, while instructors gain access to performance insights and submission trends through data visualization tools.

CodeLab Revival was built using React, Next.js, TypeScript, Tailwind CSS, FastAPI, PostgreSQL, and GitHub Actions, with a scalable architecture designed for future expansion and maintainability.
